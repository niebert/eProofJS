Welcome to e-ProofJS GitHub Pages.

The Webpage provides a generator for the iMathAS Code. This is the easiest way to create an e-Proof Enviroment for all of your e-Proofs in iMathAS. Further information about e-proof can be found at http://e-proof.weebly.com. The e-Proof-homepage provides screencast videos that explain how to create your own e-Proofs, store them in XML format.

IMathAS Background Code and Include Code

  • Backgroundcode IMathAS: Necessary for using E-Proof-Environment in IMathAS
  • e-Proof Include Code: Needed for all E-Proofs in IMathAS to include the backgroundcode into the e-Proof Question

e-Proof XML-Files

We’ve provided a IMathAS emulation for the e-proof enviroment for you to use. Go ahead and click Online-Demo to test environment in an interactive mode. You can easily check the Authoring-Mode to edit your e-Proof or create your own e-proof before providing the e-Proof to your students. You can publish an e-Proof either in IMathAS or in the emulated iMathAS environment. After publishing your e-Proof in iMathAS, you can easily assess your students work with the integrated assessment of IMathAS and use e-proofs together with other tasks and questions in iMathAS. Using the emulated version of the e-proof environment lets students explore the theorem and the proof but the scores are not stored and processing of the students activity is performed solely in the Firefox webbrowser. Your e-Proof Page content can be saved in the browsers local storage but the activity will not submitted to server nor preserved accross different computers.

Creating XML-Proofs manually

If you would like to create your e-Proofs in XML or IMathAS format save your in the Authoring enviroment. If you want to share your own proofs with the community just place the iMathAS-files NOT AS PRIVATE to your IMathAS repository so that other teacher can add or template your e-proof manually. In addition to supporting regular IMathAS content, sharing of e-Proof works best in the generated XML-Format. The Authoring Tools for the e-Proof enviroment can generated static iMathAS files from XML. Authoring Tools for the e-Proof environment make it easy to create XML-Files yourself. ASCII Math is the syntax to write Math Code´.
